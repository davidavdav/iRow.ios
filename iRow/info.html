About iRow

iRow is an App that tracks your speed, position and strokes during rowing.

Quick help

Ergometer tab

The Ergometer tab shows basic statistics while rowing: speed, distance and stroke tempo. These come in `current', `average' and `total' variants.

Start button

Pressing the start button starts a track. Total time, distance and number of strokes are reset to zero, averages start from this moment.

Current

Under `Current' you find the current speed and stroke frequency. Values in this part are always updated whenever a change in position or stroke movement is detected.

The speed is expressed in time/500m, m/s or distance/h—you can tap the speed indicator to cycle to the units of your choice (you can change to imperial units in the settings). To the right you find the current stroke frequency. The strokes are detected by accelerations in the `length' direction of your iPhone, they are detected best if you lie the device flat in the boat.

Average

Under `Average' you find average speed and stroke frequency, measured over the period since pressing the start button. Pressing the average speed indicator cycles the units in backward direction.

Total

Under `Total' you find total time, total distance and total number of strokes, since the start button was pressed. Beneath the total distance, in parenthesis, you find the iPhone's current positioning accuracy.

Map tab

The map show you where you are, and what your track so far is. You can zoom/pan like in the normal Map App.

Zoom control bezel

At the bottom is a bezel with four buttons that control the zoom mode.

  • By pressing the crosshair button, the map stays centered at the current location.
  • When the "track" button is pressed, the map is moved to show the current track.
  • By selecting "course", the map is move to the current course, if any.
  • If "none" is selected, no automatic zoom control is in effect.

Course mode

At the top is the button. By pressing this, you select the course mode. In this mode, you can:

  • Define the start/finish of a course to be rowed. Press the button and a pin is dropped at the middle of the map. You can precicely "aim" by inspecting the crosshair in the middle while adjusting the map position and zoom level.
  • Define consecutive segments of the course, by adding more pins. The total distance of the course is shown at the bottom right in a green background.
  • Move pins, by selecting them (single tap) and then drag them to adjust their position.
  • Remove pins, by selecting them, and pressing the button.
  • Delete the current course, by pressing the "course" button and shifting it sideways.

In course mode, the tracking starts automatically by passing start- and finish line. See below.

Automatic start/stop in course mode

When course mode is selected, the ergometer starts/stops automatically when crossing start and finish lines. Course mode is active if the course button is green in the Map tab:

When you move outside the course on either side and select the Ergometer tab, the start button changes to "Prepare for start". When you press the button, it changes to "Waiting to cross the start line". The total distance shows the distance to the start line.

Still to go...

When the start line is passed, the numbers in the display show estimations of how much time/distance / how many strokes are necessary to reach the finish.

Finish

When crossing the finish line in course mode, the "finish" button is automatically pressed. Totals now revert to total time/distance/strokes again, and the track recording is stopped.

Options tab

The options tab allows you to access the iRow database, change settings and browse though this help.

Track

A track is a record of a rowing trip or excercise. It stores the location, time, and stroke profile information. Additionally, you can add information about the boat used and the people in the boat.

  • In the "Save Current Track" menu you can name the track and location, and specify the boat and the people in it. You can always update or change this information later.
  • In the "Browse Tracks" menu you can select previously saved tracks, and get global statistics or view the detailed rowing speed profile.

Course

A course is a fixed route that can be rowed, which allows automatic start/stop of the track statistics. You can create a course form the Map tab by dropping pins (see above).

  • In the "Save Course" menu, you can save the course currently defined in the Map tab. You can give it a name and specify the waterway the course is located.
  • In the "Browse Courses" menu, you can inspect courses and load them for usage.

Help

In the help section you can read this help and change some global settings of iRow.

The following settings can be changed:

  • Stroke sensitivity. The stroke detection sensitivity should be set normally at about 70%—the acceleration threshold value 0.10 should show up in the title. If you experience spurious strokes, move the slider a little to the left, if your strokes are not detected well, move the slider a little to the right.
  • Unit system for speed and distance. You can choose between metric (m, km, km/h) and imperial (yd, mile, mph).
  • Speed Units. You can choose between time/500m, m/s and km/h (or mph in the case of imperial units). Note that you can also cycle through these options from the Ergometer tab view, by tapping any of the speeds shown in the display.
  • Track position while off. This allows for registering your track when you have turned the screen off, or when iRow is running in the background. Normally, when a track is recorded, the auto-lock feature of the device is disabled, so that the information on the display is shown even if you don't operate the app. If you turn the screen off manually this option might be useful, but please note that the stroke detection cannot function when another app is running in the foreground or when the screen is turned off.
  • Auto save tracks. With this switched on, a track is saved every time a track recording is finished. This can be either be because the "Finish" button is tapped, or because the finish line is crossed in course more.
  • Stroke Profile. Turning on the experimental support for stroke profile inspection allows you to inspect the acceleration profile of the boat during the track. This can be done from the Options tab / Browse Tracks / track / Inspect Track page by pressing the "switch" button. Note that this still is experimental as we need more stroke profile data from different rowers and different boats to make the stroke alignment algorithm better.
  • 100Hz sampling. This option causes the frequency of data acquisition from the accelerometers to be 100 Hz, which appears to be the hardware limit of the accelerometers in the iPhone. This may result in less noisy stroke profile recording, but may also drain the batteries quicker. Note that even with this option on, the accelerometer data is stored at 10Hz.
  • Auto orientation. This option alows you to position the device in any orientation. After a couple of strokes the direction of movement should be picked up automatically. Note that the stroke profile display still only shows the acceleration along the length-direction of the iPhone.

Rowing Mates

By keeping a small database of the people you regularly row with, you can easily specify who was participating in each track you store. This then allows for computing statistics such as total mass, available power etc. You are one of the people in the database, but you have a separate entry. While adding a person to the database, it is possible to choose someone from the Addressbook app in your iDevice.

Boats

You can also enter the boats you row in the database, and specify the name, type, weight and build date. You can indicate which boat you used when storing a track.

Import/Export

You can import tracks, courses, rowing mates and boats from iTunes or e-mail. If you connect the device to iTunes, select the decive, go to the "Apps" tab, scroll down to "File Sharing," and click the iRow icon. Under "Documents" you can then add files of the type "iRow" that you may receive from other users of the iRow app. If the files are sent to you by e-mail, you can directly open these in iRow from the e-mail App. All data base items found are shown, and you can select which ones to import in the database by tapping them. Note that importing one or several items does not link the relations in the database, e.g., when you import your friend's track that was rowed in her boat, the "boat" entry is left blank because you may not have that boat in your database.

You can export iRow database items (tracks, courses, rowing mates and boats) by tapping "Export database." You can choose between iTunes and e-mail. With iTunes, the data base files show up under "Documents" of your App's File Sharing proerties in iTunes. You can also select e-mail to send the file to yourself or a rowing mate by e-mail.

In the various sub-menus there are separate import/export options as well. In "Browse Tracks" and "Browse course" you have the option to save the track or course in Google Earth's kml format. You can export this kml by e-mail or though iTunes.

iCloud support

The database (tracks, courses, rowers and boats) is automatically synchonized with iCloud if you have activated iCloud. When you install this app on another iDevice (e.g., an iPad) the existing iCloud database is downloaded and further kept synchronized.
